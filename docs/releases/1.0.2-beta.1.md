# Advanced Multi Column — 1.0.2-beta.1

## Highlights

- Reading View marker-based rendering is now significantly more stable.
- Fixed column disappearance while scrolling in Reading View by moving rendering outside Obsidian's volatile preview sizer section list.
- Reduced large blank space before rendered content in Reading View.
- Improved compatibility with banner/frontmatter-based plugins (including pixel-banner style banner/logo rendering).

## Internal Changes

- Reworked reading-mode rendering pipeline in `src/column/reading-view.ts`.
- Added dedicated Reading View host container and lifecycle diagnostics for render/disappearance events.
- Updated active Reading View CSS layout behavior in `src/styles/layout.css`.

## Compatibility

- Minimum Obsidian version remains `0.15.0`.

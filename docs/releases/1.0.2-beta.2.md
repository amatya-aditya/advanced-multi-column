# Advanced Multi Column v1.0.2-beta.2

## Changes

### Reading View Rewrite
- Replaced CSS `:has()` driven content switching with a host-based wrapper approach.
- The wrapper is now placed as a sibling of the sizer (`.amc-reading-columns-host`), avoiding conflicts with other plugins that inject elements into the sizer.

## Technical Notes
- Reading View uses a host-based wrapper as a sibling of `.markdown-preview-sizer`. Obsidian's `el-*` section divs are hidden via CSS; all content is re-rendered into the wrapper using `MarkdownRenderer.render()`.
- Frontmatter is skipped during wrapper rendering to avoid double display.
